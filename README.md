# Таргет Про — лендинг курса

Готовый статический лендинг для бесплатного размещения на Vercel.

## Состав проекта

- `index.html` — основной сайт
- `vercel.json` — настройки для Vercel
- `README.md` — инструкция

## Как запустить на Vercel

1. Создайте новый репозиторий на GitHub.
2. Загрузите туда файлы `index.html`, `vercel.json`, `README.md`.
3. Зайдите на Vercel.
4. Нажмите `Add New Project`.
5. Выберите репозиторий.
6. Нажмите `Deploy`.

## Что поменять перед запуском

- Дату старта потока.
- Цены в блоке тарифов.
- Отзывы, если нужны реальные.
- Контакты и подключение формы заявок.
- Ссылки на Telegram / WhatsApp / Google Forms.

## Подключение заявок

Сейчас форма показывает сообщение об успешной отправке без передачи данных.
В `index.html` найдите функцию `submitForm()` и подключите нужный сервис:

- Google Forms
- Telegram Bot
- Make.com
- CRM
